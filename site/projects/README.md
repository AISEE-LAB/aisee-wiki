# 开源项目

这里收录 AISEE 团队自有或主导维护的开源项目。它和 [推荐资源](/resources/) 的边界不同：本栏目介绍团队资产、项目状态、使用入口和后续路线；推荐资源则保留为外部项目、工具和方法论资料的精选目录。

## 栏目边界

| 类型 | 放在这里 | 不放在这里 |
|---|---|---|
| 团队项目 | AISEE 团队维护或主导的 repo、插件、模板、schema pack、示例工程。 | 只是被本站推荐的外部项目。 |
| 项目介绍 | 项目定位、适用场景、成熟度、仓库入口、文档入口、路线状态。 | 未核对的安装命令、实时版本信息、外部榜单。 |
| 后续路线 | 已确认会继续建设的项目说明和待补能力。 | 普通学习资料、文章链接或工具合集。 |

## 项目清单

正式清单等待维护者补充。补充时建议每个项目至少包含：

- 项目名称和一句话定位。
- GitHub 仓库或文档入口。
- 当前状态：规划中、开发中、验证中、可用或归档。
- 适合谁使用，以及和 AISEE Plugin、OpenSpec、Compound Engineering 的关系。

## 与推荐资源的区别

[推荐资源](/resources/) 继续展示外部 AI Coding、Agent、MCP、Skill、UI/UX 和工程方法论资源。这里不做外部资源排行榜，也不复制推荐资源的分类目录。
