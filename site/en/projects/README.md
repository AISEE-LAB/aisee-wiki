# Open Source

This section is reserved for open source projects maintained or led by the AISEE team. It is separate from [Recommended Resources](/en/resources/): this section describes team-owned projects, while Recommended Resources stays focused on curated external references.

## Scope

| Type | Included here | Not included here |
|---|---|---|
| Team projects | Repos, plugins, templates, schema packs, or example projects maintained or led by the AISEE team. | External projects that are only recommended by this wiki. |
| Project notes | Positioning, maturity, repository links, documentation links, and roadmap status. | Unverified installation commands, live version claims, or external rankings. |
| Roadmap | Confirmed project direction and known gaps. | General reading material or tool collections. |

## Project List

The official list will be added after the project names, repository links, and status labels are confirmed.

Each project entry should include:

- Name and one-line positioning.
- GitHub repository or documentation link.
- Current status: planned, in development, validating, usable, or archived.
- Who should use it, and how it relates to AISEE Plugin, OpenSpec, or Compound Engineering.
