# AISEE

AISEE 是 AI-Enhanced Software Engineering 的方法与插件主题，用于把需求链路、技术上下文、change 拆分、OpenSpec 协作方式和可复用工程流程组织成一套工程机制。

## 方法范围

AISEE 关注的不是单个命令，而是从需求到验证的连续链路：

- 把模糊需求整理成可审查输入。
- 把项目事实、技术约束和风险组织成上下文。
- 把已确认目标拆成 OpenSpec change。
- 把实现、评审、验证和知识沉淀交给明确的工作流承接。
